## A fake operating system simulator that generates dad jokes for Linux, Windows, and Mac.

## Features
- Terminal-style GUI
- Dad jokes and worst humor u will ever hear
- Fake system logs

## Instalation for diffrent os
### 🐧 Ubuntu / Debian / Linux Mint

```bash
sudo apt update
sudo apt install python3 python3-tk git -y

git clone https://github.com/risotto-slackware/JokeOS.git
cd JokeOS
python3 jokeos.py

```
### Fedora
```bash

sudo dnf install python3 python3-tkinter git -y

git clone https://github.com/risotto-slackware/JokeOS.git
cd JokeOS
python3 jokeos.py
```
### RHEL / CentOS / Rocky Linux / AlmaLinux
